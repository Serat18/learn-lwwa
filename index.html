What is Lightweight Web App?

Lightweight Web App is a Web App that is Lightweight for its structure. It's a HTML file that has JavaScript, CSS, WebGL etc. all inside it with a manifest for configuration. But the HTML could be heavy, so its not so lightweight

Why was it made?

Flash games ran in a native window, but HTML5 games run in a browser window. What if HTML5 games could also run in a native window, or maybe run some other HTMLs in native window. That's why.

Can it replace others?

Yes, It can replace not too complex PWAs. Such as Chrome Calculator, its made for basic calculations so it should be a single HTML file. And also, LWWAs don't use service workers, The HTML file and manifest downloads into your browser and your browser just shows it.

{
  "name": "app",

    "128": "icon-128.png",
    "256": "icon-256.png"
  }
  *Everything is optional and there can be more icon resolutions.
}
